# Kanban_GraphQL Api
GraphQL API for Kanban boards

## Prerequisites

Before you begin, ensure you have met the following requirements:
* You have installed `Node`
* You have a database `Mysql | Postgres`

## Starting <kanban_graphql>

To migrate database :

```
npm run db:migrate:reset
```

To seed database :

```
npm run db:seed
```

To start :

```
npm run start:dev
```
